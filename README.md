SCAndSkip is a mod for GTA San Andreas Android that automatically skips the EULA and Social Club login, launching directly from the main menu.

![Demonstration](media/video.gif)

---

## ⚙️ Installation

1. Download the file `libSCAndSkip.so` or `libSCAndSkip64.so`
2. Place it in the folder: /sdcard/Android_unprotected/data/com.rockstargames.gtasa/mods/
3. Open the game. The mod will load automatically.
